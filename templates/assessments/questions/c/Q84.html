84.c언어 프로그램에서는 조건에 따라 문장을 선택해서 실행할 수 있다.
조건문이 참이라면 if 아래에 연결된 문장이 실행되며 거짓일 경우 else 아래에 연결된 문장이 실행된다.
if ( 조건문 )
printf(“참이다”);
else
printf(“다르다”);
조건문에는 다음과 같은 문장들이 올 수 있다.
a > b : a 가 b 보다 크다면 참, 아니면 거짓
a >= b : a 가 b 보다 크거나 같다면 참, 아니면 거짓
a == b : a 가 b와 같다면 참, 아니면 거짓
a != b : a가 b와 다르면 참, 아니면 거짓
