76.c언어로 만든 프로그램은 위에서 아래로 한 줄씩 실행된다.
int main(void) 후 나오는 중괄호 안에 있는 문장들이 프로그램에서 실제 실행되는 문장들이다.
return 0; 문장은 정상 종료를 의미하며 다음의 문제들을 푸는데 고려하지 않아도 된다.
c언어에서는 printf() 명령어를 사용하면 사용자에게 괄호 안의 내용을 보여줄 수 있다.
아래 프로그램의 실행 결과는 다음과 같다.

                    
#include 
int main(void){
	printf("A");
	printf("BB\n");
	printf("CCC");
	return 0;
}

출력 :
ABB
CCC
다음의 프로그램의 예상되는 출력 결과를 쓰시오. 

                
#include 
int main(void){
	printf("XXX\n");
	printf("YY\n");
	printf("ZZZ");
	return 0;
}
