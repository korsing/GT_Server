95.c언어 프로그램에서는 같은 종류의 값들을 배열의 형태로 저장할 수 있다.
int a[8] = {1, 2, 3, 4, 5, 6, 7, 8 };
(배열의이름[크기] = { 값1, 값2, … };)
위의 문장은 아래와 같은 형태로 저장된다.
배열의 각 값은 0번부터 순서대로 매겨지며 “배열의이름[순서]”으로 저장된 값을 사용할 수 있다.
다음 프로그램의 예상되는 출력 결과를 쓰시오.

int main(void){
	int a[8] = { 0, 0, 0, 0, 0, 0, 0, 0 };
	int i;
	int j = 1;
	for ( i=0 ; i<8 ; i++){
		a[i] = j;
		j = j + i;
	}
	for (i=0; i<8;i++)
		printf(“%d “,a[i]);
	return 0;
}
