78.c언어에서는 변수를 통해 특정한 값을 저장해 놓을 수 있으며 변수의 이름은 다양하게 정할 수 있다.
아래의 방법을 통해 ‘a’라는 공간에 ‘5’라는 값을 저장해 놓을 수 있다.
( ‘=’ 는 오른쪽의 값을 왼쪽으로 대입한다는 의미이다.)
1) int a = 5;
2) int a;
a = 5;
아래의 프로그램의 결과는 다음과 같다.

            
#include 
int main(void){
	int a = 5;
	printf(“%d\n”, a);
	printf(“%d”, a);
	return 0;
}
출력 :
5
5

다음의 프로그램의 예상되는 결과를 쓰시오. 

            
#include 
int main(void){
	int a = 5;
	int b = 10
	printf(“%d %d”, a, b);
	return 0;
}
