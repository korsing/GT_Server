79.c언어 프로그램에서는 scanf() 명령어를 사용해 사용자에게 특정 값을 입력 받을 수 있다.
다음 문장을 실행하면 a라는 공간에 사용자의 입력(정수)을 저장할 수 있다.
int a;
scanf(“%d”, &a);
다음 프로그램을 실행하고 사용자가 정수 100을 입력했을 때, 예상되는 출력 값을 쓰시오. 

                
#include 
int main(void){
	int a;
	scanf(“%d”, &a);
	printf(“%d%d”, a,a);
	return 0;
}
