85.c언어 프로그램에서는 동일한 문장을 반복해서 수행을 할 수 있는 반복문이 있다.
for ( 초기문 ; 조건문 ; 증감문 ){
실행될 문장
}
1) 초기문은 반복문이 실행되기 전 한번만 실행된다.
2) 조건문이 참이라면 중괄호로 묶여있는 문장들이 순서대로 실행된다. 거짓이라면 반복문을 벗어나 다음 문장들을 순서대로 실행하게 되며 반복문은 종료된다.
3) 조건문이 참이고 중괄호로 묶인 문장들을 전부 실행했다면 증가 또는 감소 시키는 증감문이 실행된다.
4) 다시 2)으로 돌아간다.
다음 프로그램은 5번을 반복해서 “안녕하세요.” 를 출력하는 프로그램이다. 10번을 반복해서 출력하고 싶다면 조건문을 어떻게 바꾸어야 하는지 쓰시오.
( ++ 연산자는 해당 변수의 값을 1 증가시킨다.)
( -- 연산자는 해당 변수의 값을 1 감소시킨다.)
