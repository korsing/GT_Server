a + b : a와 b 값을 더한다.
a – b : a와 b 값을 뺀다.
a * b : a와 b 값을 곱한다.
a / b : a를 b로 나눈 몫을 구한다.
a % b : a를 b로 나눈 나머지를 구한다.
a = b : b 값을 a에 대입한다.
